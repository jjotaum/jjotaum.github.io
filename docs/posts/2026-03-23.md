# Introducing Bolt

Bolt is my most recent submission to the MacOS AppStore, a fast and beautifully simple tool designed to measure your internet performance without unnecessary complexity. Built with 100% native code, it delivers accurate results while remaining extremely lightweight and efficient.

## Why I created it?

There have been many occasions when I’ve found myself needing to quickly check my internet connection speed. In the past, I used web tools from companies dedicated to running these speed tests, basically the usual websites we all know, which track our information and are often filled with ads. For me, that made the experience far from ideal.

Later, I started using the CLI that Apple introduced in recent versions of its system, which eliminated ads and the need to open a browser. Overall, it worked very well for checking my connection via the terminal, but at times I felt it lacked a simple, native feeling UI that would make the experience more user friendly, and that’s how Bolt was born.
