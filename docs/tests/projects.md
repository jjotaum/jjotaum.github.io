A swift package to facilitate Apple Music API integration for iOS, MacOS, tvOS & watchOS projects.
Link: https://github.com/jjotaum/AmuseKit

# 👨‍💻 ChromaCLI

Link: https://github.com/jjotaum/SwiftyTree
